# PumpScope Demo

PumpScope is a build-in-public project for the Pump.fun Hackathon. 
It tracks early-stage token dev activity, analyzes wallet transactions, 
and provides risk signals for users.

## Features (Demo)
- Track developer wallet purchases and sales
- Visualize token market cap changes
- Show risk signals and holder analysis

## How It Works
- Currently a demo HTML page
- Displays example tables and images
- Later will support live tracking and automated data

## Purpose
This project demonstrates how we can monitor early-stage tokens, 
provide transparency, and empower users with real-time insights.
